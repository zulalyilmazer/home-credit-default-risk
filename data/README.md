# Data Directory

The raw dataset for this project is too large to be hosted on GitHub (~2.68 GB). 
You can download the official dataset directly from the Kaggle competition page:

🔗 [Home Credit Default Risk Dataset](https://www.kaggle.com/competitions/home-credit-default-risk/data)

### Setup Instructions:
1. Download the zip file from the link above.
2. Extract all CSV files (e.g., `application_train.csv`, `bureau.csv`, etc.).
3. Place them locally inside this `data/` folder to run the local notebooks.
